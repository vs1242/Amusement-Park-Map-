Help

Welcome to Amusement Park Navigator. This application was created to help navigate Six Flags Over Texas.

To use this app, find the Ride List Page, or find the settings/map pages. When using the Ride List page, there are sort and filter options for you to use. Once you find a ride that you want to navigate to, simply hit the "Directions" button and the map will take you right to it.

To use the settings and map pages, you can select which types of rides you would like to see on the map using the settings page. After doing that, clicking on any pin will show you the names of the rides and give you the option to navigate to that ride. Setting changes will automatically update on the map.
